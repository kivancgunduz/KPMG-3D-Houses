# KPMG-3D-Houses
Visualization Houses from the Specific Address in Belgium as a 3D Model for [KPMG](https://home.kpmg/be/en/home.html) managed by [BeCode](https://becode.org/)

#### Installation
```
- Python 3.9.5 (64bit)
- Numpy 1.22.1
- Pandas 1.4.0
- Matplotlib 3.5.1
```
